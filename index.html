《心理抚养》读后感

发表于 2021-08-09 更新于 2021-09-24 分类于 读书笔记
本文字数： 阅读时长 ≈ NaN:aN

作者简介

作者李玫瑾，中国人民公安大学犯罪心理学教授。从事犯罪心理学、未成年犯罪等研究，目前在中国人民公安大学任教。

总的收获概括

  1. 很多人成年后的问题，都与早期的抚养息息相关，这是有规律的。
  2. 早期的抚养，0-3岁建立情感依赖，3-6岁培养初期性格， 小学毕业之前都是培养心理依恋的阶段，以6岁之前尤为重要。
  3. 早期最好父母自己带，或者尽可能的陪伴孩子，有稳定的抚养过程，从而建立起深厚的心理依恋，这是教育孩子的心理资本。没有这个资本，孩子，尤其是青春期过后的孩子，单纯的管教很难起作用，
    他基本不听你的。不但不听，很多时候反而起反作用，造成关系决裂甚至反目成仇。
  4. 性格培养、规则观念等需要在3岁后就尽早有意识培养。青春期时再教就晚了。挫折教育、社会性教育、怎样与他人相处。
  5. 不要太担心批评或者惩罚会有不良影响，一定程度上可以培养孩子心理耐受性，防止说不得现象。只是主要要恩威并施、鼓励关爱为主，注意分寸。
  6. 多带着运动。多受一些身体的苦，对培养心理的强大有积极作用。对减少抑郁、释放不良情绪等都有好处。
  7. 孩子的问题要改，多半都是大人的问题，看看大人哪里需要改。
  8. 青春期管孩子，方法需要变。不要多话，点到为止。

所以有时觉的在目前自己所处的行业，选择了相对轻松的公司，尽可能的每天能多陪孩子，而没有去寻求996，拿高薪，还是值得的。想想如果每天虽然都能回家，但孩子基本每天都见不到你，那跟不在又有什么区别呢，又谈何建立起深厚的心理依恋资本呢？

《欲罢不能：刷屏时代如何摆脱行为上瘾》

发表于 2020-09-18 更新于 2021-09-24 分类于 读书笔记

作者：亚当·奥尔特
书名：《欲罢不能：刷屏时代如何摆脱行为上瘾》
版本：北京华章图文信息有限公司, 2018. Kindle edition

不是自己自制力薄弱，而是敌人太强大

问题并不出在人缺乏意志力上，而在于“屏幕那边有数千人在努力工作，为的就是破坏你的自律”

当前的人们经常会沉迷在各种刷屏行为中，短视频，微博，信息流等等，有时一刷几个小时都停不下来。但刷屏时很快乐，结束之后人们往往又开始自责，为什么自己的自制力这么低呢。其实不是自己自制力低，而是敌人太强大。

行为上瘾是什么

社交媒体对孩子的伤害

在和孩子进行互动交流时，要专注、认真。不要敷衍！

父母在和婴儿进行互动时，婴儿本能地会追随父母的眼睛，并且关注父母的表情，情绪的变化。一个互动时分心的家长会培养出分心的孩子，因为如果家长都不能集中注意力，就会把相同的关注模式教给孩子。

人类通过观察他人的行为，学习共情和体谅。没有直接反馈，共情得不到发展，而这又是一项发育极为缓慢的技能。

《拆掉思维里的墙》

发表于 2020-07-22 更新于 2021-09-24 分类于 读书笔记
本文字数： 阅读时长 ≈ NaN:aN

关于书和作者

作者古典，原新东方的老师，之后从事心理咨询，后又从事职业发展咨询事业。这本书之前看过，但是没什么印象，但是几年之后再看一遍时，却感觉受益良多。好书还是应该多看几遍，不同的生活阶段会有不同的收获。

一些总结

关于安全感

  1. 很多打着为你好的旗帜，对你所做的行为，其实都是因为他们自身缺乏安全感。比如爱情中的我离不开你，这些不是爱，而是恐惧。这个世界没有谁离不开谁。
  2. 人们有时会错把依赖当成爱。爱不是死死抓住，而是要放开。
  3. 安全感别人给不了你，只能自己给自己。很多人在爱情中说对方给自己给不了安全感，但其实依靠别人给予的安全感，永远不会是安全的。

关于恐惧

有的人似乎什么都可以做好，兴趣广泛，特长众多。而有的人似乎什么都不会，什么都不敢做。人和人之间的智力差距其实很小，这种情况的区别在于后者经常会给自己设置限制。这些人常常恐惧一些问题，比如这个我不敢做，那个害怕做不好，等等。但恐惧本身很奇怪，当你翻来覆去在脑海中想时，他会越来越强大，但当你豁出去，迈开步子真的去做的时候，它就会缩成一团，悄悄溜走。回过头来，你会发现恐惧的很多事情，大多数情况下根本不会发生。

勇于尝试，不要给自己轻易设限，用我不会，我做不好多丢人等一些借口来搪塞自己。比如自己学习做饭的过程就是一个体会。
